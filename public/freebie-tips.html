The fixes

The calm fixes

  1. Inbox ≠ storage. If you need it later, file it. If it needs action, task it. If not, delete.
  2. Create a 5-folder spine. 01_Admin, 02_Active, 03_Reference, 04_Records, 99_Archive.
  3. Use dates in filenames. YYYY-MM-DD at the start makes sorting and searching sane.
  4. Stop version chaos. One “current” file, one owner. Everyone else reads, not rewrites.
  5. Write the subject line properly. “Action required by Friday: …” beats “Hi”.
  6. Use search, not scrolling. Learn 3 searches you use daily and save them.
  7. Kill noisy notifications. Keep only: direct messages, calendar reminders, and urgent mentions.
  8. Keep a one-page runbook. “Where files live, who owns what, how to update.”
  9. Weekly 15-minute reset. Clear downloads, process inbox, rename 5 files properly.
  10. Make “done” visible. A short list of what's finished prevents circular emails and doubt.
