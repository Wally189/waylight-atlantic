Starter Website route

Templates

This page is the concrete version of the first website offer on the Services & Pricing page: Starter Website. If you want a calm, dependable web presence without commissioning a bespoke build, this is the quickest route in.

The fundamental design stays as it is. In most cases that means three simple pages on a fixed framework, with the personal project option kept deliberately lean at two pages. Images, text, contact details, and light colour direction can be tailored to suit.

If you need more than that, you are no longer in template territory. Wider layout changes, extra page architecture, feature work, or heavier redesign should move to one of the other website options on the pricing page.

Ordered scope

A fixed, practical foundation

The value of the starter route is not endless flexibility. The value is that the structure is already disciplined, readable, and ready to adapt without drifting into unnecessary complexity.

What stays fixed What can be tailored If you need more
Compact page structure Most routes keep a simple three-page format; the personal project route is a deliberate two-page exception in the same compact spirit. Move to Website Improvement, Tailored Basic, or Tailored Standard if the page architecture itself needs to change.
Stable navigation and layout logic Your own text, practical calls to action, contact details, imagery, and light colour direction can be applied within the template frame. If navigation, sections, or the visual system need reworking, that becomes a different website plan rather than a template tweak.
Simple maintainable delivery Each template launch includes Care & Control so the site can be kept current without a fragile setup. Feature additions, integrations, or materially wider scope are quoted separately under the broader options on the pricing page.

Short version: the starter offer stays simple on purpose. The template is the foundation; your content is the layer that changes.

How it works From starter package to live website

  1. The client selects the Starter Website route as the entry package.

  2. A host and domain then need to be chosen. Waylight prefers Netlify for this kind of static site because it stays simple and dependable, but another suitable setup can be used if needed.

  3. If Waylight is arranging the setup, this hosting and domain stage is the main initial direct charge. After that, the template itself is prepared within the agreed starter scope.

  4. Once the setup is in place, the client chooses the images, text, contact details, and other in-scope content adjustments for the selected template.

  5. The site is then prepared, reviewed, and published on the client's own hosting and domain.

  6. After launch, Care & Control begins so the site can be maintained without the structure drifting.

Scope boundary

What stays inside the starter route

Within the Starter Website route Needs one of the other pricing options
Swap in your own images, graphics, and practical brand cues Commissioning a different page layout, section architecture, or more bespoke design system.
Rewrite text, service summaries, welcome copy, contact details, and calls to action within the existing structure Major content expansion, new page types, or additional page counts beyond the published starter frame.
Choose a restrained colour direction that suits the organisation while keeping the build calm and maintainable Fundamental changes to navigation, functionality, forms, integrations, or the underlying build approach. That sits under a different plan on the pricing page.

Why this boundary exists: the template route is meant to stay fast, dependable, and low-friction. Personalisation is welcome; redesign belongs in the wider website plans.

Working method

Simple in the right order

  1. Choose the template that is already closest to your actual use case.

  2. Send over your text, images, contact details, and preferred colour direction.

  3. Waylight applies that material within the fixed starter structure and prepares the launch under Care & Control.

  4. If the work turns into a redesign or a larger build, the scope moves openly to one of the broader website options on the pricing page.
